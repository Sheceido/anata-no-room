- 🧰 How it's Made 🛠️ -

The scene was crafted in two parts:

  • 3D models of each object was created using Blender, with each object's texture/coloration unwrapped and then "baked" onto a 4096 x 4096 jpeg file, while the wall-posters and TV Screen were rescaled and mapped separately for correct dimensions/orientation

  • UV mapping of each geometry that corresponds to the jpeg file textures are saved within the exported .glb model file from blender.

  • The ThreeJS javascript library did a lot of heavy lifting: used to import 3D assets, textures, as well as adding particle textures, combining HTML and webGL elements, texturizing a video source onto a 3D screen object for a working TV, a load manager for load screens, and interactive elements with camera dollying/rotating/zooming.
